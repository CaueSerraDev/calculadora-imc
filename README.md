# Calculadora de IMC

Projeto simples em HTML, CSS e JavaScript para cálculo do Índice de Massa Corporal (IMC).

## Fórmula utilizada

IMC = peso / (altura * altura)

## Classificação

- Abaixo de 18.5 → Abaixo do peso
- 18.5 a 24.9 → Peso ideal
- 25 a 29.9 → Sobrepeso
- 30 a 34.9 → Obesidade grau I
- 35 a 39.9 → Obesidade grau II
- Acima de 40 → Obesidade grau III

## Tecnologias usadas

- HTML
- CSS
- JavaScript
